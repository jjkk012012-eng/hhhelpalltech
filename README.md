# Factory Radar Simple v10

검색어와 상세 주소만 입력해 주변 제조 공장을 찾고, 지도 표시·이동시간·편도/왕복/주문배송 예상비를 계산하는 정적 웹앱 프로토타입입니다.

## 실행

압축을 풀고 `index.html`을 브라우저에서 엽니다.

Google Maps API 키가 기본 입력된 버전입니다. 우측 상단 **지도 API 설정**에서 호출 제한 수를 조정할 수 있습니다.

## 이번 버전의 핵심 개선

- 전체 277,532건을 처음부터 로딩하지 않습니다.
- 기준 주소에서 시군구를 인식하고, 해당 지역 데이터 조각만 로딩합니다.
- `판금`, `CNC`, `금형`, `도금`, `사출` 등 생산품·원자재·업종명·회사명·생산품 카테고리를 검색합니다.
- `판금` 검색 시 생산품에 `판금`이 들어간 공장을 최우선으로 점수화합니다.
- 임의 좌표로 공장 핀을 찍지 않습니다. API로 좌표가 확인된 공장만 지도에 표시합니다.
- Google Maps 키가 있으면 Distance Matrix로 실제 도로 거리/시간을 우선 계산합니다. 없으면 근사 시간/비용을 계산합니다.

## 사용 예

- 검색어: `판금`
- 기준 주소: `남양주시`
- 결과: 남양주시 공장 데이터만 로딩하여 판금 관련 공장을 표시합니다.

- 검색어: `CNC`
- 기준 주소: `화성시 동탄`
- 결과: 화성시 데이터에서 CNC/머시닝/절삭/정밀가공 관련 공장을 검색합니다.

## 주의

원본 엑셀에는 위도·경도 컬럼이 없습니다. 정확한 지도 표시는 주소 API 지오코딩 결과에 따라 달라집니다.


## v10 변경점

- API 호출 상한 설정 추가
- 검색당 좌표 변환 최대 건수 기본 20건
- 검색당 도로 거리/시간 계산 최대 20건
- 좌표/거리 캐시를 v10 전용으로 분리
- API 설정 가이드 `API_SETUP.md` 추가
- 전체 데이터는 포함하되, 브라우저에서는 기준 지역 조각만 로딩

정확한 지도 표시를 위해서는 우측 상단 `지도 API 설정`에서 Google 또는 Naver 키를 입력하세요.


## v11 변경

- Google Maps API 키 기본 적용
- Google 지오코딩 요청에 대한민국 제한 조건 적용
- 검색당 좌표 변환/도로시간 계산 제한은 유지
- 좌표/거리 캐시는 브라우저 localStorage에 저장

운영 전에는 Google Cloud Console에서 HTTP referrer 제한, API 제한, 예산 알림을 꼭 설정하세요.
